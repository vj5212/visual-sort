About Visual Sort

Visual Sort is a visualizer for some of the most fundamental sorting algorithms: Bubble Sort, Merge Sort, and Quick Sort. The application is a static website written entirely in Javascript, HTML, and custom CSS. It is designed to clearly visualize the way that each algorithm attempts to sort unordered arrays/lists by showing the swapping of elements that takes place in the algorithm. There also information about the time complexity of each algorithm and external sources to help apply these algorithms in code.
